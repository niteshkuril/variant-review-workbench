## 🗂️ What is variant-review-workbench?

variant-review-workbench helps you analyze genetic variant files quickly. It focuses on ClinVar data to help you triage and report Variants Call Format (VCF) files. You can also add information from PharmGKB to enrich your results. This tool is useful if you work with genetic data, but you do not need to be a programmer to use it.

---

## 📋 Key Features

- Opens and reads VCF files from genetic studies  
- Highlights ClinVar-important variants for easier review  
- Adds supplementary info from PharmGKB databases  
- Creates clear, organized reports you can save and share  
- Works on any typical Windows computer  
- Simple interface that guides you step-by-step

## 🤝 How to Use Variant-Review-Workbench Step-by-Step

### 1. Load VCF File

- Click on the **Open File** button.  
- Find your VCF file on your computer.  
- Select it and wait for it to load in the software.

### 2. Review Variants

- After loading, you will see a list of variants.  
- Variants linked to ClinVar will be highlighted.  
- You can click each variant to see detailed information.  

### 3. Add PharmGKB Information (Optional)

- Click the **PharmGKB Enrichment** option.  
- The software connects to the PharmGKB database online.  
- It pulls related data about drug responses or pharmacogenomic effects.  
- The added info appears alongside variant details.

### 4. Apply Filters

- Use filtering tools to narrow down variants by significance, gene, or other criteria.  
- Filters help you focus on the most important results.

### 5. Generate Reports

- Click the **Generate Report** button after reviewing variants.  
- Choose if you want the report as PDF or Excel.  
- Save the report to your preferred folder.

## 🛠️ Technical Details

variant-review-workbench is built with Python. It reads VCF files and uses ClinVar annotations as the primary source of variant data. It connects to PharmGKB to provide additional pharmacogenomics context.

The software is focused on simplicity while offering useful data to those working with genetic variants. No coding knowledge is required.
